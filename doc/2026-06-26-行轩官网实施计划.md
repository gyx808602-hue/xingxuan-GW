# 行轩官网实施计划

## 1. 实施目标

在空目录中快速初始化一个基于 React 的科技风官网项目，并完成首页静态展示页开发与构建验证。

## 2. 技术选型

- 构建工具：Vite
- 前端框架：React
- 语言：JavaScript
- 样式：原生 CSS

## 3. 文件规划

| 模块层级 | 文件路径 | 修改逻辑简述 |
| :--- | :--- | :--- |
| Frontend | `F:/xingxuan/package.json` | 初始化项目依赖与脚本 |
| Frontend | `F:/xingxuan/index.html` | 站点入口 HTML |
| Frontend | `F:/xingxuan/src/main.jsx` | 挂载 React 应用 |
| Frontend | `F:/xingxuan/src/App.jsx` | 首页主结构组件 |
| Frontend | `F:/xingxuan/src/styles.css` | 官网整体样式与响应式适配 |
| Doc | `F:/xingxuan/doc/项目进展.md` | 记录阶段性执行情况 |

## 4. 分步任务

### 任务 1：初始化 React 项目

- 使用 Vite 创建 React 项目骨架
- 安装依赖并确认开发脚本可用

### 任务 2：拆分首页结构

- 在 `App.jsx` 中完成首页主结构
- 模块包括：
  - Header
  - Hero
  - Capabilities
  - Solutions
  - Brand Value
  - Contact/Footer

### 任务 3：实现科技风视觉样式

- 在 `styles.css` 中完成：
  - 深色渐变背景
  - 卡片网格布局
  - 按钮与描边效果
  - 移动端响应式适配

### 任务 4：验证构建结果

- 运行依赖安装
- 运行生产构建
- 检查构建是否成功

## 5. 验证步骤

- 安装依赖：`npm install`
- 运行构建：`npm run build`
- 预期结果：Vite 构建成功，生成 `dist` 目录，无报错退出
